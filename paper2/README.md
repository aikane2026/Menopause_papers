## Overview

This repository contains the code to reproduce all analyses and figures in the paper.  The model formalises the Phase 2 signature hypothesis: the human-specific steepening of follicle depletion after age 35 is a predicted signature of grandmothering selection acting on the Phase 2 atresia rate.  Selection coefficients on Phase 2 rate alleles are computed from the quantitative life-history model described in the companion paper (under review).

## Data

No new empirical data are generated by this paper.  All input values are taken from published sources cited in the manuscript:

- Follicle-depletion parameters: Faddy et al. (1992) *Hum. Reprod.* 7, 1342–1346
- Human–chimpanzee comparison: Cloutier et al. (2015) *GeroScience* 37, 557–569
- CHEK2 variant frequencies and effect sizes: cited in the manuscript (refs [15,16])
- Fitness-gain parameters: companion paper (under review)

## Reproducing the figures

```bash
python3 atresia_model.py        # Figure 1  ->  fig_atresia_model.png
python3 make_fig2_phase2.py     # Figure 2  ->  Figure_2_phase2_force.png
python3 chek2_figure.py         # Figure 3  ->  Figure_3_CHEK2.png
```

Each script writes its figure to the working directory.  Runtime: under 1 minute on any standard laptop.
